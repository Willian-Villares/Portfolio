# Portfólio

Este é um projeto pessoal do meu **Portfólio**.

Este é um projeto que tem a proposta de apresentar e aprimorar os meus conhecimentos e práticas de **Design** e **Código**.

## Sobre o Projeto

> O objetivo do projeto é criar um local para exibir meus trabalhos e compartilhar informações sobre mim e meu perfil profissional.

**Algumas tecnologias utilizadas e recursos introduzidos**

- HTML, CSS, JavaScript
- Adobe XD / Figma
- Normalize CSS
- Wireframe 
- Flexbox
- Etc...
